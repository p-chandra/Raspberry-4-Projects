# With Hypervisor
Common reason to go with the hypervisor is when needing to install multiple Operating System due to having **mixed-criticality systems**

| System Function   | OS Type              |
| ----------------- | -------------------- |
| Flight control    | safety RTOS          |
| Navigation        | ARINC partitioned OS |
| Cockpit UI        | Linux                |
| Maintenance tools | Linux                |
